Mô tả và Giới thiệu
  • Tuỳ chỉnh phông chữ đồng hồ trên màn hình khoá của bạn
  • Cách dùng như sau:
  • Sao chép file font .ttf của bạn vào đường dẫn / Library/ Application Support/ ClockFonts
  • Với ios 15 đường dẫn sẽ là /var/mobile/ClockFonts (Nếu không có thư mục ClockFonts thì bạn hãy tạo nó thủ công bằng Filza) Sau đó Respring thiết bị.
  • Nếu font đồng hồ không thay đổi hãy thử file font .ttf khác. Hỗ trợ font có định dạng .ttf .ttc .otf .otc
  • Lưu ý: Tên font nên được ghi bằng số( VD: 90.ttf) để chỉnh cỡ cho font chữ. Chỉ đặt 1 file font duy nhất trong /Library/Applicatios Support/ClockFonts để tránh lỗi
